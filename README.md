# 📚 Repositório de Exercícios - Curso Completo de Desenvolvimento Web


## 📋 Sobre o Repositório

Bem-vindo ao repositório completo de exercícios e materiais didáticos do curso de **Desenvolvimento Web Full Stack**. Este repositório contém uma coleção abrangente de conteúdos estruturados para o aprendizado progressivo de tecnologias web modernas.

### 🎯 Objetivo

Fornecer um material completo, organizado e prático para estudantes e profissionais que desejam dominar o desenvolvimento web, desde os fundamentos de HTML/CSS até aplicações completas com Node.js e bancos de dados.

---

## 🗂️ Estrutura do Repositório

```
repo_exercicios/
├── 📁 banco_de_dados/          # Curso completo de PostgreSQL (60h)
├── 📁 moduloHTMLCSS/            # HTML5 e CSS3 + Projetos

├── 📁 moduloBootstrap/          # Bootstrap (15 aulas)
├── 📁 moduloTailwind/           # Tailwind CSS
├── 📁 legado/                   # Materiais de referência
└── 📁 modelo/                   # Templates base
```

## 📚 Módulos Disponíveis

### 🗄️ 1. Banco de Dados - PostgreSQL

**Pasta:** `banco_de_dados/`

**Descrição:** Curso intensivo completo de Banco de Dados Relacional com PostgreSQL.

#### 📖 Conteúdo

- ✅ **15 dias de conteúdo (60 horas)**
- ✅ **55 exercícios** progressivos com gabarito completo
- ✅ **10 aulas** estruturadas em arquivos .txt
- ✅ Plano de aula detalhado
- ✅ Scripts SQL prontos para executar
- ✅ Dados CSV para importação (clientes, produtos, pedidos)

#### 📂 Arquivos Principais

- `README.txt` - Guia completo do curso
- `plano_aula_bd_postgres.txt` - Cronograma de 15 dias
- `exercicios_bd_postgres.txt` - 55 exercícios organizados
- `gabarito_bd_postgres.txt` - Respostas completas
- `scripts.sql` - Todos os scripts SQL
- `GUIA_INSTALACAO.txt` - Instalação PostgreSQL + DBeaver
- `INSTRUCOES_IMPORTACAO_CSV.txt` - Como importar dados

#### 🎓 Tópicos Abordados

1. Conceitos básicos e DDL
2. Tipos de dados e constraints
3. Relacionamentos e foreign keys
4. DML e transações
5. Consultas e filtros (WHERE, LIKE, BETWEEN)
6. JOINs (INNER, LEFT, RIGHT, FULL)
7. Funções agregadas e GROUP BY
8. Subconsultas e CTEs
9. Views e índices
10. Funções, procedures e triggers

#### 🛠️ Ferramentas

- PostgreSQL 14+
- DBeaver Community

## 🚀 Como Usar Este Repositório

### Para Estudantes

#### 1️⃣ **Clone o repositório**

```bash
git clone <url-do-repositorio>
cd repo_exercicios
```

#### 2️⃣ **Escolha um módulo**

Comece pelo que mais se adequa ao seu nível:

- **Iniciante**: `moduloHTMLCSS/` → `modulo_JS/introducao/`
- **Intermediário**: `modulo_JS/` → `moduloNode/`
- **Avançado**: `banco_de_dados/` → `moduloNode/10_toughts/`

#### 3️⃣ **Siga a estrutura**

Cada módulo tem sua própria organização e READMEs internos.

#### 4️⃣ **Pratique!**

- Execute os exemplos
- Faça os exercícios
- Consulte os gabaritos apenas após tentar
