# Modulo React

Este modulo concentra projetos e exercicios praticos com React.

## Estrutura Atual

- `hotel-react/` - Projeto de site de hotel com React, Vite e React Router.

## Projeto Incluido: hotel-react

### Conteudos praticados

- Componentes funcionais
- Hooks (`useState`, `useEffect`)
- Roteamento com `react-router-dom`
- Formularios controlados
- Organizacao de estilos em arquivos CSS por secao

### Como executar

1. Acesse a pasta do projeto:

```bash
cd hotel-react
```

2. Instale as dependencias:

```bash
npm install
```

3. Rode em desenvolvimento:

```bash
npm run dev
```

4. Gere build de producao (opcional):

```bash
npm run build
```
