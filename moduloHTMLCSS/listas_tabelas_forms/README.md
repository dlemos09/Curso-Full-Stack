# Listas, Tabelas e Formulários

Conteúdo organizado para exemplos de HTML.

## Estrutura

- `formulario/`: exemplos de formulários (login, campos, semântica, meter/progress, busca).
- `tabelas/`: exemplo de tabela com cabeçalho, corpo e rodapé.
- `details.html`: uso de `<details>` e `<summary>` para mostrar/ocultar conteúdo.
- `listasAninhadas.html`: listas aninhadas (`ul`) e listas de definição (`dl`).

## Como visualizar

Abra qualquer arquivo `.html` no navegador ou via VS Code (simple browser). Os exemplos são simples e auto-contidos.
